## Notes Page (editable by Teacher)
Class Schedule page (editable by Teacher)
Discussion Room page
Score/Evaluation page (editable by Teacher)

## Teacher Portal:

Notes Page (editable by Teacher)
Class Schedule Page (editable by Teacher)
